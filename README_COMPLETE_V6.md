# FootballIQ Complete v6

This is the feature-complete MVP built from the uploaded FootballIQ project.

## Included
- Email/password authentication and persistent sessions
- Username onboarding and profile
- XP, rating, daily streak, accuracy, levels and leaderboard
- 10 Football Duels packs
- Higher or Lower
- Who Am I?
- Career Path
- Referee Decisions
- Would You Scout Him?
- Daily five-question challenge
- Saved prediction picks
- Duplicate XP prevention per quiz ID
- Supabase master SQL matched to this code
- FootballIQ branding throughout the active app

## Install
1. Run `FootballIQ_SUPABASE_COMPLETE_V6.sql` in Supabase SQL Editor.
2. Keep the existing Vercel environment variables:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. Replace the old local project with this folder, or copy these files into it.
4. Run:
   - `npm install`
   - `npm run build`
5. Commit and push to the existing `footballiq` GitHub repository.

## Important content note
The product and progression systems are functional. Some active-player statistics and the demo prediction fixtures should be refreshed/verified before a large public launch.
