# FootballIQ V3 Visual Identity Update

This build adds a distinct visual identity to each major game mode while preserving the existing quiz logic, authentication and Supabase integration.

## Added
- Referee Arena: dark floodlit stadium atmosphere with yellow/red-card accents.
- Scout Vision: tactical pitch, heat-map glow and recruitment-room styling.
- Who Am I: purple mystery identity styling.
- Career Path: visual timeline/archive identity.
- Higher or Lower: orange streak-arena identity.
- Daily Challenge: calendar and streak-focused presentation.
- Prediction Centre: match-centre data visual identity.
- Redesigned quiz library with visually distinct mode cards.
- Multi-tab competitive leaderboard for Overall, Referees, Scouts, Duelists, Detectives, Predictors and Daily streaks.
- Direct category leaderboard links from each game mode.
- Mobile-responsive decorative visuals and reduced visual clutter on smaller screens.

## Verification
`npm run build` completed successfully with Next.js 16.2.6.

## Install
1. Replace your current project with this folder, or copy the changed source files into your repository.
2. Restore your own `.env.local` values from your deployed project.
3. Run `npm install`.
4. Run `npm run dev` to test locally.
5. Commit and push to GitHub for Vercel deployment.

The `.env.local` file is intentionally excluded from the delivery ZIP to protect your Supabase credentials.
